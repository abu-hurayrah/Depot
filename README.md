# Depot

<div align="center"><h2><br/><br/>
    A data logger made with ALNOOR<br/><br/>Made by Raiyaxi Ziaoi
</h2></div>
<br>

Usage:

```java
var log = new Depot (
        "log.txt",
        "log",
        "LOG: ",
        "dd-MM-yyyy HH:mm:ss"
    );
log.log (
    "Hello World"
);
```
